# DgeaHeatmap

<!-- badges: start -->
<!-- badges: end -->

The goal of DgeaHeatmap is to enable data extraction from nanostring
GeoMx DSP raw data. Furthermore, to enable easy heatmap building and
differential gene expression analysis.

## Installation

You can install the development version of DgeaHeatmap from
[GitHub](https://github.com/) with:

``` r
# install.packages("devtools")
devtools::install_github("leolanci/Dgea_Heatmap_Package")
```
